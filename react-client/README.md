# Cliente HTTP para React

Librería que provee funcionalidades para axios, entre ellas se destacan:

  - Interceptors para Requests: Se incluye funcionalidades para gestionar   Token con JWT y manejar errores del cliente. 
  - Interceptors para Response: Se incluye funcionalidades para gestionar enviar peticiones y manejar errores del servidor.
  - Custom hooks: Manejo de lógica para hacer peticiones y gestionar los resultados desde los componentes.
